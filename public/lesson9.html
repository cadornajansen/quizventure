Lesson 9: Arrays & Collections

Arrays and collections are powerful tools for storing and managing groups of data in Java. Arrays are simple and fixed-size, perfect for known quantities, while collections like ArrayList are dynamic and adaptable. Think of arrays as a fixed shelf and collections as an expandable backpack!

Arrays: Fixed-Size Data Storage

Arrays store multiple values of the same type in a contiguous block of memory. They're declared with a type and size, and accessed via indices starting from 0.

  • Declaration: int[] numbers = new int[5]; or int[] numbers = {1, 2, 3};
  • Access: numbers[0] gets the first element.
  • Length: numbers.length gives the size.
  • Looping: Use for loops or enhanced for-each loops to iterate.

Fun Fact: Arrays are great for game inventoriesâ€”store item IDs or scores in a neat list!

Basic Array Example

Here's how to create and use an array:

int[] numbers = {1, 2, 3};  // Fixed-size array
System.out.println(numbers[0]);  // Access first item: 1
System.out.println("Array length: " + numbers.length);  // 3

// Looping through array
for (int num : numbers) {
  System.out.println("Number: " + num);
}

Collections: Dynamic Data Structures

Collections are part of java.util and provide flexible storage. ArrayList is a resizable array that can grow or shrink dynamically.

  • Import: import java.util.ArrayList;
  • Declaration: ArrayList<Integer> list = new ArrayList<>();
  • Methods: add() to insert, get(index) to access, size() for length, remove() to delete.
  • Advantages: No fixed size, easy to modify.

Fun Fact: Collections are like magic bagsâ€”add or remove items on the fly, perfect for dynamic leaderboards!

ArrayList Example

Using ArrayList for flexible storage:

import java.util.ArrayList;

ArrayList<Integer> scores = new ArrayList<>();
scores.add(100);  // Add elements
scores.add(200);
scores.add(150);

System.out.println("First score: " + scores.get(0));  // 100
System.out.println("Size: " + scores.size());  // 3

// Looping through ArrayList
for (int score : scores) {
  System.out.println("Score: " + score);
}

Quest: Build an Array of High Scores

Let's create a high scores array for your leaderboard! We'll store top scores and display them.

public class HighScores {
  public static void main(String[] args) {
    int[] highScores = {1500, 1200, 1800, 900, 2100};  // Array of scores
    
    System.out.println("Top High Scores:");
    for (int i = 0; i < highScores.length; i++) {
      System.out.println((i + 1) + ". " + highScores[i]);
    }
    
    // Find the highest score
    int max = highScores[0];
    for (int score : highScores) {
      if (score > max) max = score;
    }
    System.out.println("Highest Score: " + max);
  }
}
// Output:
// Top High Scores:
// 1. 1500
// 2. 1200
// 3. 1800
// 4. 900
// 5. 2100
// Highest Score: 2100

Exercise 2: ArrayList for Dynamic Scores

import java.util.ArrayList;

public class ScoreListDemo {
  public static void main(String[] args) {
    ArrayList<Integer> playerScores = new ArrayList<>();
    playerScores.add(500);
    playerScores.add(750);
    playerScores.add(600);
    
    System.out.println("Player Scores:");
    for (int i = 0; i < playerScores.size(); i++) {
      System.out.println("Score " + (i + 1) + ": " + playerScores.get(i));
    }
    
    // Add a new score
    playerScores.add(800);
    System.out.println("Updated size: " + playerScores.size());
  }
}
// Output:
// Player Scores:
// Score 1: 500
// Score 2: 750
// Score 3: 600
// Updated size: 4

Common Pitfalls and Tips

Watch for ArrayIndexOutOfBoundsException when accessing beyond array length. Use generics with collections (e.g., ArrayList<String>) for type safety. Arrays are faster for fixed data, collections for flexibility. Practice sorting or searching arrays for advanced use!

Pro Tip: In game development, use arrays for fixed grids (e.g., map tiles) and ArrayLists for player inventories that change.
