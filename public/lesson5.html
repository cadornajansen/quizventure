Lesson 5: Conditional Statements

If statements check conditions and execute code accordingly. Perfect for game logic, like "if health > 0, continue!" They evaluate boolean expressions and branch your program based on true/false results. Without them, code would be rigid and unable to respond to variables or inputs.

Basic If Statement

Executes code only if the condition is true.

if (condition) {
  // Code to run if true
}

If-Else Statement

Adds an alternative path if the condition is false.

if (condition) {
  // True block
} else {
  // False block
}

If-Else If-Else Chain

Handles multiple conditions sequentially.

if (condition1) {
  // Block 1
} else if (condition2) {
  // Block 2
} else {
  // Default block
}

Tip: Always use curly braces for clarity.

Logical Operators

Combine conditions: && (and), || (or), ! (not).

  • &&: Both must be true.
  • ||: At least one true.
  • !: Negates the condition.

Nested Conditionals

Place if statements inside others for complex logic.

Examples and Code Snippets

Conditionals in game scenarios.

// Basic if
if (age >= 18) {
  System.out.println("You're an adult adventurer!");
} else {
  System.out.println("Train more!");
}

// With operators
if (health > 50 && hasSword) {
  System.out.println("Ready for battle!");
}

Try It Yourself: Conditional Exercises

Practice with these examples.

Exercise 1: Basic If-Else

public class IfElseDemo {
  public static void main(String[] args) {
    int score = 75;

    if (score >= 60) {
      System.out.println("Passed!");
    } else {
      System.out.println("Failed. Try again!");
    }

    // Nested example
    boolean bonus = true;
    if (score >= 60) {
      if (bonus) {
        System.out.println("Bonus points awarded!");
      }
    }
  }
}
// Output:
// Passed!
// Bonus points awarded!

Exercise 2: Logical Operators

public class LogicalOpsDemo {
  public static void main(String[] args) {
    int health = 40;
    boolean shield = true;

    if (health > 30 && shield) {
      System.out.println("Safe to proceed.");
    } else if (health > 30 || shield) {
      System.out.println("Caution advised.");
    } else {
      System.out.println("Danger! Retreat!");
    }

    // Not operator
    if (!shield) {
      System.out.println("No shield equipped.");
    }
  }
}
// Output:
// Safe to proceed.

Common Pitfalls and Tips

Avoid dangling else (use braces). Test edge cases. In games, use conditionals for win/lose states or item checks.

Pro Tip: Combine with loops for dynamic quests.
