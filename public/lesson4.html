Lesson 4: Control Flow

Control flow is the "director" of your programâ€”it decides what happens next based on conditions, like choosing paths in a quest. Use loops and conditionals to make decisions or repeat actions. Master this, and your code will level up! Without control flow, programs would run linearly, unable to adapt to inputs or states.

Conditional Statements

Conditionals execute code based on true/false conditions. Use them for branching logic.

  • if: Executes if condition is true.
  • if-else: Adds an alternative path.
  • if-else if-else: Multiple conditions.
  • switch: Efficient for multiple fixed values.

Loops

Loops repeat code until a condition is met. Essential for iteration.

  • for: Best for known iterations (e.g., counting).
  • while: Repeats while condition is true.
  • do-while: Executes at least once, then checks.

Examples and Code Snippets

Basic control flow in action.

// If-else example
if (health > 0) {
  System.out.println("Continue playing!");
} else {
  System.out.println("Game over!");
}

// For loop example
for (int i = 1; i <= 5; i++) {
  System.out.println("Level " + i);
}

Try It Yourself: Control Flow Exercises

Experiment with these to understand flow.

Exercise 1: Conditional Statements

public class ConditionalsDemo {
  public static void main(String[] args) {
    int score = 85;

    if (score >= 90) {
      System.out.println("Grade: A");
    } else if (score >= 80) {
      System.out.println("Grade: B");
    } else {
      System.out.println("Grade: C or lower");
    }

    // Switch example
    int day = 3;
    switch (day) {
      case 1: System.out.println("Monday"); break;
      case 2: System.out.println("Tuesday"); break;
      case 3: System.out.println("Wednesday"); break;
      default: System.out.println("Other day");
    }
  }
}
// Output:
// Grade: B
// Wednesday

Exercise 2: Loops

public class LoopsDemo {
  public static void main(String[] args) {
    // For loop
    for (int i = 0; i < 3; i++) {
      System.out.println("For loop iteration: " + i);
    }

    // While loop
    int count = 0;
    while (count < 3) {
      System.out.println("While loop count: " + count);
      count++;
    }

    // Do-while loop
    int num = 0;
    do {
      System.out.println("Do-while num: " + num);
      num++;
    } while (num < 3);
  }
}
// Output:
// For loop iteration: 0
// For loop iteration: 1
// For loop iteration: 2
// While loop count: 0
// While loop count: 1
// While loop count: 2
// Do-while num: 0
// Do-while num: 1
// Do-while num: 2

Common Pitfalls and Tips

Avoid infinite loops (ensure exit conditions). Use break/continue for control. In games, conditionals handle events like "if enemy defeated, level up."

Pro Tip: Combine loops and conditionals for complex logic, like iterating through player inventory.
